## Blockquote

Use the `>` syntax to create a styled citation. Add an author with a line starting by `—`.

::::preview
:::code-preview

> Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium.
>
> — John Doe

:::

:::content-preview

```mdx
> Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium.
>
> — John Doe
```

:::
